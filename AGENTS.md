# AGENTS.md

## Purpose

This repo is an HTML-first learning workspace for passing the Korean Electrical Engineer practical exam.

## Required Workflow

1. Read `MISSION.md`, `LEARNING_STATE.md`, `RESOURCES.md`, `NOTES.md`, and recent `learning-records/` before teaching.
2. If the mission or learner level is unclear, ask one short question first.
3. Create one small lesson or reference at a time.
4. Save source HTML lessons in `lessons/`.
5. Save source review cheat sheets in `reference/`.
6. Copy only public-safe HTML to `docs/` for GitHub Pages.
7. Update `LEARNING_STATE.md` and `sync/CHATGPT_PROJECT_UPDATE.md` after meaningful changes.

## HTML-First Publishing Rules

- The default output is an HTML lesson.
- Markdown lessons are fallback files or helper files for ChatGPT conversation only.
- Company and mobile study should use the HTML published through GitHub Pages.
- Use `docs/` as the GitHub Pages public site.
- Keep authoring copies in `lessons/*.html` and `reference/*.html`.
- Copy or sync only public-safe HTML into `docs/lessons/` and `docs/reference/`.
- Electrical Engineer practical-exam lessons are built around five past-exam-centered problems per day.
- Each lesson should include button-style quizzes and immediate feedback.

## Safety Rules

- Do not commit paid PDFs, work documents, personal information, or raw copied source material.
- Keep originals in ignored local folders only.
- In `RESOURCES.md`, record only source name, purpose, and private/local location description.
- Never include full PDF originals, paid textbook originals, or full copied past-exam question sets in the repository or `docs/`.
- Public HTML may include only problem summaries, solution flow, formulas, wrong-answer points, and self-made variant questions.
- Do not create a real problem-based lesson until the user provides source material or a safe summary.

## Style

- Korean by default.
- Short, conclusion-first responses.
- Prefer practical examples and retrieval practice.
- Avoid large course generation before the mission is clarified.
