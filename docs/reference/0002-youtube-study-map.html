Workflow

하루 5문제에 강의를 끼우는 순서

  1. 먼저 문제 5개를 풉니다. 강의는 풀이 전에 길게 보지 않습니다.
  2. 틀린 문제마다 이론 축을 1개 붙입니다.
  3. 위 표에서 해당 축의 첫 자료를 1개만 고릅니다.
  4. 시청 후 외울 문장 1개, 조건 판별 1개, 자작 변형문제 1개로 바꿉니다.
  5. HTML lesson에는 강의 원문이 아니라 내 풀이 흐름과 오답 포인트만 남깁니다.
