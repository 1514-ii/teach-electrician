# Mission: 전기기사 실기 합격

## Why

전기기사 실기 시험 합격을 목표로, 기출 중심의 반복 학습을 HTML lesson으로 축적한다. 회사와 모바일에서는 GitHub Pages로 공개 가능한 HTML만 보고, ChatGPT 대화는 보조 설명과 복습용으로 사용한다.

## Success looks like

- 하루 5문제 기준으로 기출 유형을 복습한다.
- 각 문제를 원문 암기가 아니라 풀이 흐름, 공식, 오답 포인트로 설명할 수 있다.
- 공개 가능한 HTML lesson을 GitHub Pages에서 바로 확인한다.
- 틀린 문제는 오답 포인트와 자작 변형문제로 다시 연습한다.
- PDF 원문, 유료교재 원문, 기출문제 원문 전체를 Git에 커밋하지 않는다.

## Constraints

- 기본 산출물은 `lessons/*.html` HTML lesson이다.
- Markdown은 ChatGPT 대화용 보조 산출물로만 사용한다.
- 공개 HTML에는 문제 요약, 풀이 흐름, 공식, 오답 포인트, 자작 변형문제만 포함한다.
- 원문 PDF, 유료교재, 전체 기출문제 원문, 업무자료, 개인정보는 repo와 `docs/`에 포함하지 않는다.
- 자료가 들어오기 전에는 실제 lesson을 만들지 않는다.

## Out of scope

- PDF 원문 저장소화
- 유료교재나 기출문제 원문 전체 재게시
- 시험 범위를 벗어난 전기 일반 상식 강의
- 자료 없이 실제 문제 기반 lesson을 임의 생성
