# RESOURCES.md Format

`RESOURCES.md` is the curated source list for this workspace. Lessons should rely on these resources instead of memory alone.

```md
# {Topic} Resources

## Knowledge

- [Resource title](https://example.com)
  What it covers. Use for: the specific lesson types it supports.

## Wisdom

- [Community or mentor source](https://example.com)
  Why it is useful. Use for: real-world feedback, judgment, or practice review.

## Private or Local Source Notes

- `{Material name}`: used for `{purpose}`. Stored in `{private/local location}`. Do not commit original.

## Gaps

- {Missing resource area}
```

Rules:

- Prefer official, primary, or highly trusted sources.
- Annotate every resource with purpose and use.
- Do not commit paid PDFs, private work material, or personal source documents.
- Keep original materials in ignored local folders only.
- Remove shallow or unreliable resources.
