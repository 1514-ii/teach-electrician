# Notes

## Teaching Preferences

- Korean is preferred unless the learner asks otherwise.
- Keep answers short and easy to scan.
- Start with the conclusion.
- Give one recommended next action before alternatives.
- Use simple wording when possible.

## Source Safety

- Do not commit paid PDFs, private work documents, personal data, or raw source archives.
- Record source names and usage notes in `RESOURCES.md` instead.
