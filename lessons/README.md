# HTML Lesson 생성 규칙

전기기사 실기 학습의 기본 산출물은 `lessons/*.html`이다.

## 생성 기준

- 하루 5문제 기준으로 만든다.
- 기출 중심으로 구성한다.
- 실제 자료가 제공되기 전에는 문제 기반 lesson을 만들지 않는다.
- 문제 원문 전체를 복사하지 않는다.
- PDF 원문, 유료교재 원문, 업무자료, 개인정보를 포함하지 않는다.

## 파일명

```text
lessons/0001-topic-name.html
```

예시:

```text
lessons/0001-transformer-basic-5.html
```

## Lesson 구성

각 HTML lesson은 아래 순서를 기본으로 한다.

1. 오늘의 5문제 범위
2. 문제별 공개 가능한 요약
3. 풀이 흐름
4. 사용 공식
5. 오답 포인트
6. 자작 변형문제
7. 버튼형 퀴즈
8. 즉시 피드백
9. 다음 복습 지시

## 공개 동기화

- 작성본은 `lessons/*.html`에 둔다.
- 공개 가능한 lesson만 `docs/lessons/*.html`로 복사한다.
- 공개 HTML에는 문제 요약, 풀이 흐름, 공식, 오답 포인트, 자작 변형문제만 넣는다.
- 원문 확인이 필요한 내용은 `RESOURCES.md`에 자료명/용도/위치 설명만 기록한다.

## 금지

- 기출문제 원문 전체 게시
- 유료교재 본문 복사
- PDF 원문 커밋
- 회사/개인정보 포함
- 출처가 불명확한 해설을 사실처럼 단정
