Reference 0001

실기 이론 지도

오장육부와 최근 기출 자료를 바탕으로 만든 공개용 학습 지도입니다. 원문 문제와 PDF 본문은 포함하지 않고, 하루 5문제 lesson을 만들 때 연결할 이론 축만 정리합니다.

Rule

문제보다 먼저 분류

문제를 보면 먼저 송배전, 조명, 수변전, KEC, 시퀀스 중 어디인지 표시합니다.

Flow

공식보다 조건

공식을 외우기 전에 단상/3상, 선간/상전압, 역률, 기준용량 같은 조건을 확인합니다.

Review

오답은 변형문제로

틀린 문제는 같은 숫자로 다시 풀지 않고 조건 하나를 바꾼 자작 변형문제로 복습합니다.

Daily Lesson Template

하루 5문제 연결법

Step 1

5문제 고르기

최근 기출에서 연속 5문제를 고르되, 너무 한 축에 몰리면 1문제는 약점 축으로 교체합니다.

Step 2

이론 축 연결

각 문제를 위 7개 축 중 하나에 붙이고, 필요한 공식과 조건을 짧게 정리합니다.

Step 3

즉시 피드백

버튼형 퀴즈로 조건 판별, 공식 선택, 오답 포인트를 바로 확인합니다.

First Route

첫 학습 순서

  1. 2024년 1회 기출에서 1-5번을 첫 lesson 후보로 잡습니다.
  2. 각 문제를 이론 축에 분류합니다.
  3. 오장육부에서 연결되는 공식/개념을 찾아 풀이 흐름으로 재작성합니다.
  4. OCR 기출집에서 같은 유형의 과거 반복 출제를 확인합니다.
  5. 공개 HTML에는 원문이 아니라 요약, 흐름, 공식, 오답 포인트, 자작 변형문제만 남깁니다.

Quick Check

분류 연습

문제에서 “선로 길이, 역률, 수전단 전압, 전압강하”가 같이 보이면 먼저 어느 축으로 분류해야 할까요?

Source Safety
